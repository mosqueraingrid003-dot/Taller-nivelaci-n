15. ¿Cuáles son los tipos de datos primitivos en Java?
Java define ocho tipos de datos primitivos, que son los más básicos y no son objetos. Se dividen en: 
Enteros: byte, short, int y long.
Punto flotante (decimales): float y double.
Carácter: char (representa un solo carácter).
Lógico: boolean (almacena solo true o false). 

16. ¿Cómo funcionan las estructuras de control de flujo en Java?
Estas estructuras permiten alterar el orden de ejecución de un programa: 
if / else: Evalúan una condición booleana para decidir si se ejecuta un bloque de código u otro.
switch: Selecciona un camino entre múltiples opciones basándose en el valor de una variable.
Bucles (while, for): Ejecutan tareas repetidamente mientras se cumpla una condición específica. 

17. ¿Por qué es importante usar nombres significativos para variables y métodos?
El uso de nombres descriptivos facilita la legibilidad y el mantenimiento del código. Permite que otros programadores (o tú mismo en el futuro) comprendan el propósito de cada elemento sin necesidad de análisis profundos, reduciendo confusiones y errores. 

18. ¿Qué es la Programación Orientada a Objetos (POO)?
Es un paradigma de programación que organiza el software en torno a "objetos" en lugar de solo funciones o lógica. Un objeto combina datos (variables/atributos) y comportamientos (funciones/métodos) que interactúan entre sí. 

19. ¿Cuáles son los cuatro pilares de la Programación Orientada a Objetos?
Los conceptos fundamentales para crear sistemas robustos y reutilizables son: 
Abstracción: Seleccionar solo los aspectos relevantes de un objeto para el sistema.
Encapsulamiento: Ocultar el estado interno y proteger los datos de accesos indebidos.
Herencia: Crear nuevas clases basadas en clases ya existentes.
Polimorfismo: Permitir que un mismo mensaje o método se comporte de distintas formas según el objeto. 

20. ¿Qué es la herencia en POO y cómo se utiliza en Java?
Es el mecanismo por el cual una clase (hija o subclase) deriva de otra (padre o superclase), adoptando sus métodos y propiedades para reutilizar código. En Java, se implementa mediante la palabra reservada extends y solo permite herencia simple (una clase solo puede heredar de una única superclase). 

21. ¿Qué son los modificadores de acceso y cuáles son los más comunes en Java?
Son palabras clave que establecen el nivel de accesibilidad o visibilidad de clases y sus miembros. Los más comunes son: 
public: Acceso total desde cualquier clase.
protected: Acceso dentro del mismo paquete y por subclases.
private: Acceso restringido únicamente a la clase donde se declaró.
Default (sin modificador): Acceso solo para clases dentro del mismo paquete. 

22. ¿Qué es una variable de entorno y por qué son importantes?
Son valores de configuración externos al código que afectan el comportamiento de aplicaciones y del sistema operativo. Son cruciales porque permiten cambiar configuraciones (como rutas de instalación o credenciales de red) sin necesidad de modificar el código fuente, facilitando la portabilidad del software entre diferentes entornos.
